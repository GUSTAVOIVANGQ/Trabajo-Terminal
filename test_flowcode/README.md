# Visualizador de Símbolos ISO 5807

Esta es una aplicación demostrativa desarrollada en Flutter que renderiza gráficamente los símbolos de diagramas de flujo definidos en la norma **ISO 5807-1985**.

## Objetivo

El propósito de esta aplicación es servir como referencia visual e interactiva para la documentación y estudio de símbolos de diagramación, mostrando la capacidad de Flutter para el dibujo personalizado (`CustomPainter`).

## Símbolos Incluidos

La aplicación categoriza y dibuja los siguientes tipos de símbolos:
*   **Data Symbols** (Datos)
*   **Process Symbols** (Procesos)
*   **Line Symbols** (Líneas de flujo)
*   **Special Symbols** (Conectores y anotaciones)

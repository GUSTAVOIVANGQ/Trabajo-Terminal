### Para Profesores

**Sugerencias de ejercicios basados en plantillas:**

#### Ejercicio 1: Variaciones de Suma (Plantilla #1)
- Resta de dos números
- Multiplicación
- División (con validación de división por cero)
- Área de un rectángulo
- Conversión de temperaturas (°C a °F)

#### Ejercicio 2: Decisiones Simples (Plantilla #2)
- Mayor de dos números
- Positivo, negativo o cero
- Aprobar o reprobar (nota >= 60)
- Descuento según edad (niño, adulto, senior)
- Clasificación de triángulos

#### Ejercicio 3: Bucles (Plantilla #3)
- Suma de N números
- Factorial de un número
- Tabla de multiplicar
- Adivinar número con intentos limitados
- Validar entrada hasta que sea correcta

#### Ejercicio 4: Menús (Plantilla #4)
- Calculadora con 4 operaciones
- Sistema de notas (CRUD)
- Conversor de unidades
- Juego de preguntas con opciones
- Sistema de login con reintentos

#### Ejercicio 5: Documentación (Plantilla #5)
- Tomar cualquier diagrama y agregar comentarios
- Explicar cada paso del algoritmo
- Documentar variables y sus propósitos
- Agregar advertencias y notas

#### Ejercicio 6: Modularización (Plantilla #6)
- Separar validación de procesamiento
- Crear funciones para cálculos repetitivos
- Diseñar un sistema con múltiples subprocesos
- Refactorizar diagrama complejo en módulos

## 🚀 Consejos de Uso

### Para Obtener el Máximo Beneficio

1. **No te saltes plantillas**
   - Sigue el orden sugerido
   - Cada plantilla construye sobre la anterior

2. **Modifica y experimenta**
   - Cambia valores y condiciones
   - Agrega nuevos nodos
   - Prueba diferentes flujos

3. **Genera el código**
   - Siempre genera el código C
   - Observa cómo se traduce cada símbolo
   - Compara diferentes enfoques

4. **Documenta tus cambios**
   - Usa comentarios (como plantilla #5)
   - Explica por qué haces algo
   - Mantén nombres claros

5. **Valida siempre**
   - Usa el validador antes de generar código
   - Corrige errores y advertencias
   - Aprende de los mensajes

## ❓ FAQ

**P: ¿Puedo modificar las plantillas?**  
R: ¡Sí! Las plantillas son solo puntos de partida. Puedes cargarlas, modificarlas y guardarlas con otro nombre.

**P: ¿Se pueden combinar conceptos de varias plantillas?**  
R: ¡Por supuesto! Por ejemplo, puedes crear un menú (plantilla 4) que use bucles (plantilla 3) y subprocesos (plantilla 6).

**P: ¿Las plantillas se borran si borro mis diagramas?**  
R: No. Las plantillas están protegidas y siempre estarán disponibles.

**P: ¿Puedo crear mis propias plantillas?**  
R: Actualmente las plantillas son predefinidas, pero puedes guardar tus diagramas como referencia personal.

**P: ¿Qué plantilla es más importante?**  
R: Todas son importantes, pero la #1 (Suma) es fundamental para entender el flujo básico.
